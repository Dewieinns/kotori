##########################
Data acquisition with Bash
##########################
.. highlight:: bash

There are various ways of transmitting telemetry data from the command line.

Please read how to :ref:`transmit data using mosquitto_pub <daq-mqtt-bash>`
or how to do :ref:`daq-curl`.

See also :ref:`Sawtooth signal from bash <sawtooth-bash>` and
:ref:`Sawtooth signal using curl <sawtooth-curl>` for transmitting a dynamic,
slowly oscillating sawtooth signal for getting started with an example.
